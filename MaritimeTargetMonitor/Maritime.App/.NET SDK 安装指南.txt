# .NET SDK 6.0 安装指南

## 系统要求

- **操作系统**: Windows 11 64位
- **内存**: 至少 2 GB RAM
- **磁盘空间**: 至少 2 GB 可用空间
- **管理员权限**: 需要管理员权限才能安装

## 步骤一：下载 .NET SDK 6.0

1. **打开浏览器**
   - 启动任意网页浏览器（Chrome、Edge、Firefox等）

2. **访问下载页面**
   - 打开以下网址：
   - `https://dotnet.microsoft.com/download/dotnet/6.0`

3. **选择下载版本**
   - 在"SDK"部分，找到"Windows"选项
   - 选择"x64"架构
   - 点击"下载"按钮

4. **保存安装包**
   - 选择保存位置，建议保存到：
   - `C:\Users\15666\Desktop\MaritimeTargetMonitor\MaritimeTargetMonitor\Maritime.App`
   - 文件名应该类似于：`dotnet-sdk-6.0.418-win-x64.exe`

## 步骤二：安装 .NET SDK 6.0

1. **运行安装程序**
   - 找到下载的安装包文件
   - 双击运行它
   - 如果出现"用户账户控制"提示，点击"是"

2. **按照安装向导操作**
   - 点击"下一步"
   - 接受许可条款，点击"下一步"
   - 选择安装位置（默认即可），点击"下一步"
   - 点击"安装"
   - 等待安装完成
   - 点击"关闭"

## 步骤三：验证安装

1. **打开命令提示符**
   - 按下 `Win + R` 组合键
   - 输入 `cmd`，点击"确定"

2. **验证 .NET SDK 安装**
   - 在命令提示符中输入：
   ```cmd
   dotnet --version
   ```
   - 按下 Enter 键

3. **检查输出**
   - 如果看到版本号（例如：`6.0.418`），表示安装成功
   - 如果出现错误信息，表示安装失败

## 步骤四：运行 Maritime Target Monitor

1. **打开项目目录**
   - 导航到：
   - `C:\Users\15666\Desktop\MaritimeTargetMonitor\MaritimeTargetMonitor\Maritime.App`

2. **启动应用程序**
   - 双击 `启动程序.bat` 文件
   - 或双击 `Maritime.App.exe` 文件（如果已生成）

3. **如果未生成可执行文件**
   - 打开命令提示符
   - 导航到项目目录：
   ```cmd
   cd C:\Users\15666\Desktop\MaritimeTargetMonitor\MaritimeTargetMonitor
   ```
   - 构建项目：
   ```cmd
   dotnet build MaritimeTargetMonitor.sln -c Debug -r win-x64
   ```
   - 运行应用程序：
   ```cmd
   cd Maritime.App\bin\Debug\net48\win-x64
   Maritime.App.exe
   ```

## 步骤五：解决常见问题

### 问题：命令提示符中找不到 dotnet 命令

**解决方案：**
1. 检查环境变量是否正确设置
2. 重启计算机
3. 手动添加环境变量：
   - 右键点击"此电脑" → "属性"
   - 点击"高级系统设置"
   - 点击"环境变量"
   - 在"系统变量"中找到"Path"
   - 点击"编辑"
   - 点击"新建"
   - 输入：`C:\Program Files\dotnet`
   - 点击"确定"保存

### 问题：构建失败

**解决方案：**
1. 确保 .NET SDK 安装正确
2. 检查项目文件是否完整
3. 尝试清理并重新构建：
   ```cmd
   dotnet clean MaritimeTargetMonitor.sln
   dotnet build MaritimeTargetMonitor.sln
   ```

### 问题：应用程序启动失败

**解决方案：**
1. 检查配置文件 `config.json` 是否存在且格式正确
2. 检查依赖项是否完整
3. 查看日志文件获取详细错误信息
4. 尝试以管理员身份运行
